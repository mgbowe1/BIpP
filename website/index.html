BIpP

Broadcasting Ip Protocol

About BIpP

BIpP is a new take on an old networking paradigm, made more interesting through the use of public key - private key encryption. We use a server to pass messages as a hub, but only the intended recipient can decrypt the message, ensuring anonimity. This means we do not need to know the ip address of a computer to connect to it, just it's key. We can do this from anywhere in the world by hosting servers for this protocol on various internet-facing locations.
